# Comment contribuer ?
1. Forkez ce dépôt.
2. Créez une branche pour votre modification (`git checkout -b ma-modification`).
3. Commitez vos changements (`git commit -m "Description des changements"`).
4. Poussez votre branche (`git push origin ma-modification`).
5. Ouvrez une *pull request* depuis votre *fork* vers ce dépôt.
